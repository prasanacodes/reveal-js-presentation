Black-Scholes Model

Calculating the theoretical price of options:

\[ C(S, t) = N(d_1)S - N(d_2)Ke^{-r(T-t)} \]

Where:
\[ d_1 = \frac{\ln(S/K) + (r + \frac{\sigma^2}{2})(T-t)}{\sigma\sqrt{T-t}} \]

Speaker Note: Explain that S is current stock price, K is strike price, r is risk-free rate.

Calculating Compound Interest


def calculate_compound_interest(principal, rate, time):
    """
    Formula: A = P(1 + R/100)^t
    """
    amount = principal * (pow((1 + rate / 100), time))
    return amount

# Example usage
p = 10000
r = 5.25
t = 5

print(f"Final Amount: {calculate_compound_interest(p, r, t)}")
